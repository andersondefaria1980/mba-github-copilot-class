# Development Prompts

This folder contains the 11 sequential prompts to build the Customer Registration System.

## 📋 Execution Order

Execute the prompts in the following order:

1. **[prompt-01-initial-structure.md](prompt-01-initial-structure.md)** - Creates monorepo structure and scripts
2. **[prompt-02-backend-setup.md](prompt-02-backend-setup.md)** - Configures Node.js/Express backend
3. **[prompt-03-database.md](prompt-03-database.md)** - Creates PostgreSQL schema and seed
4. **[prompt-04-backend-dockerfile.md](prompt-04-backend-dockerfile.md)** - Backend Dockerfile
5. **[prompt-05-jwt-authentication.md](prompt-05-jwt-authentication.md)** - Implements JWT authentication
6. **[prompt-06-customers-crud.md](prompt-06-customers-crud.md)** - Customers CRUD
7. **[prompt-07-backend-tests.md](prompt-07-backend-tests.md)** - Jest unit tests
8. **[prompt-08-docker-compose.md](prompt-08-docker-compose.md)** - Docker orchestration
9. **[prompt-09-frontend-setup.md](prompt-09-frontend-setup.md)** - React/Vite setup
10. **[prompt-10-react-components.md](prompt-10-react-components.md)** - React components
11. **[prompt-11-frontend-docker.md](prompt-11-frontend-docker.md)** - Frontend Dockerfile + Nginx

## 🚀 How to Use

1. Open each file in order
2. Copy the prompt content
3. Paste into GitHub Copilot Chat or AI tool
4. Wait for implementation
5. Move to the next prompt

## ⚠️ Important

- **DO NOT skip prompts** - each depends on the previous
- **Execute sequentially** - do not parallelize
- Verify each step is complete before proceeding
